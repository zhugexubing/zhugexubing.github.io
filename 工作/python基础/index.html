python基础

除法（/）永远返回一个浮点数。如要使用 floor 除法 并且得到整数结果（丢掉任何小数部分），你可以使用 // 运算符；要计算余数你可以使用 %：

17 / 3 # 一般除法返回浮点数 5.666666666666667

17 // 3 # floor 除法丢掉分数部分 5 17 % 3 # % 操作符来取余数 2

交互模式中，最近一个表达式的值赋给变量 _。这样我们就可以把它当作一个桌面计算器，很方便的用于连续计算，例如: >>> price * tax 12.5625 >>> price + _ 113.0625 >>> round(_, 2) 113.06

\ 可以用来转义引号： >>> ‘doesn\’t’ # 使用转义符
“doesn’t”

如果你不想 \ 开头的字符被解释为特殊字符的话，你可以在 原始字符串 第一个引号前加 r ：

print(‘C:\some\name’) # 这里的 \n 意味着换行 C:\some ame print(r’C:\some\name’) # 注意单引号前的 r C:\some\name
